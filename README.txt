To run the web app on your pc follow these steps:
Step1:  Install python in your system.
Step2:  After installing python install streamlit using 
	pip install streamlit
	to download the necessary libraries.
Step3:	Read the requirements.txt to download the required libraries. 
Step4:  Go to folder containing the main.py file shown in filepath.png 
        and copy the path of main.py
Step5:  Open command prompt in the folder(containing main.py) 
        or use command prompt to navigate as shown in final.png image.
Step6:	Run command streamlit run main.py as shown in final.png image.
